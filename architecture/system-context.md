# 1. Концепция продукта

## 1.1 Описание продукта

Платформа представляет собой сервис для создания, подбора и публикации образов (аутфитов), где пользователь может как самостоятельно собирать стиль, так и вдохновляться готовыми образами других людей.

Продукт сочетает в себе:
- подбор одежды;
- социальную механику;
- рекомендации;
- систему публикации аутфитов.

---

## 1.2 Целевая аудитория

Основная аудитория — молодые люди 16–30 лет, интересующиеся модой, стилем и визуальным контентом.

---

# 2. Основные разделы платформы

## 2.1 Free Mode — конструктор образов

Главная интерактивная страница, где пользователь самостоятельно собирает аутфиты.

---

### 2.1.1 Как это работает

Пользователь:
- открывает раздел;
- выбирает категорию одежды;
- свайпает вещи вправо/влево;
- комбинирует понравившиеся элементы;
- собирает полноценный образ.

Механика напоминает:
- Tinder — по свайпам;
- Pinterest — по визуальному вдохновению;
- Fashion-конструктор — по созданию образов.

---

### 2.1.2 Возможности Free Mode

#### Свайп одежды

Пользователь может:
- лайкать вещи;
- пропускать;
- сохранять элементы;
- комбинировать их между собой.

---

### 2.1.3 Категории

Отдельные разделы:
- верх;
- низ;
- обувь;
- аксессуары;
- очки;
- сумки;
- украшения и др.

---

### 2.1.4 Создание собственного аутфита

После успешного подбора пользователь может:
- сохранить образ;
- опубликовать его;
- добавить название;
- добавить описание;
- добавить стиль;

---

## 2.2 Outfit Feed — лента готовых образов

Социальная лента с аутфитами пользователей.

Главная идея — бесконечный поток образов для вдохновения.

---

### 2.2.1 Механика ленты

Лента работает по принципу:
- TikTok;
- Instagram Reels;
- Pinterest Feed.

Пользователь листает аутфиты вертикально.

---

### 2.2.2 Карточка аутфита

Карточка содержит:
- список вещей;
- автора;
- стиль;
- лайки;
- сохранения.

---

### 2.2.3 Взаимодействие с контентом

Пользователь может:
- лайкать аутфит целиком;
- лайкать отдельные вещи;
- сохранять образ;
- переходить к элементам одежды;
- подписываться на авторов.

---

### 2.2.4 Алгоритм рекомендаций

Лента подстраивается под:
- любимые стили;
- лайки;
- сохранения;
- бренды;
- цвета;
- просмотренные образы.

---

# 3. Профиль пользователя

## 3.1 Страница пользователя

На странице пользователя:
- опубликованные аутфиты;
- избранное;
- подписчики;
- подписки;
- любимые бренды;
- статистика лайков.

---

## 3.2 Публикации

Каждый пользователь может:
- вести собственную fashion-страницу;
- выкладывать образы;
- оформлять визуальный стиль профиля.

---

# 6. Уникальность проекта

Главная особенность — сочетание:
- fashion-конструктора;
- соцсети;
- системы рекомендаций;
- интерактивного подбора одежды.
