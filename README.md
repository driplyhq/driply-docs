# Driply Docs

Это репозиторий для документации проекта Driply. Мы используем простую структуру с `index.html` файлами, чтобы всё это красиво отображалось через GitHub Pages (или локально), как портал-база знаний.

## Как добавить новый документ

Парни, всё просто. Хотите добавить документ?

1. **Закиньте файл:** Положите свой `.md` или `.pdf` в нужную папку (`frontend/`, `backend/` или `architecture/`).
2. **Картинки (если есть):** Если в доке есть схемы или скриншоты — создайте папку `img` рядом со своим файлом и ссылайтесь на них как обычно (`![схема](img/schema.png)`).
3. **Добавьте ссылку:** Откройте `index.html` в этой же папке и добавьте новую строчку-ссылку на свой файл в блок `<ul>`. Например:
   `<li><a href="my-awesome-doc.md">Описание архитектуры платежей</a></li>`

Пушим в мастер — через минуту всё обновится на сайте. Никаких сложных генераторов статики, просто файлы и ссылки.
