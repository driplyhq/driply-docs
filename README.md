# Driply Docs

База знаний проекта. Все доки здесь.

### Как добавить документ

1. Положите файл в нужную папку: `frontend`, `backend` или `architecture`.
2. Если нужны картинки, создайте рядом папку `img` и кладите их туда.
3. Откройте `index.html` в этой же папке. Добавьте ссылку на свой файл в список `<ul>`.

После пуша в `main` всё обновится на сайте автоматически.
